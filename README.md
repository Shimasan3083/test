# Touch Comet Chase

画面をタッチして遊ぶ、ブラウザ向けの2Dハイスコアゲームです。
プレイヤーは彗星を動かして光る星を集め、赤い隕石を避けながら高得点を目指します。

## 開き方

### いちばん簡単な方法

1. このフォルダの中にある `index.html` を探します。
2. `index.html` をダブルクリックします。
3. ブラウザ（Chrome、Edge、Safariなど）でゲーム画面が開いたら成功です。

### VS Codeを使う方法

1. VS Codeでこのフォルダを開きます。
2. 左のファイル一覧から `index.html` を右クリックします。
3. 「Open with Live Server」がある場合は、それを選びます。
4. ブラウザでゲーム画面が開きます。

Live Serverを入れていない場合は、`index.html` をダブルクリックする方法で遊べます。

### ターミナルで開く方法

このリポジトリのフォルダで、次のコマンドを実行します。

```bash
python3 -m http.server 4173
```

そのあと、ブラウザで次のURLを開きます。

```text
http://127.0.0.1:4173/index.html
```

ターミナルを止めるときは、ターミナル上で `Ctrl + C` を押します。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 画面をタップしてゲームを開始します。
3. 画面に指を置いたまま動かすと、彗星が指の位置に向かって移動します。
4. 黄色い星を取るとスコアが増えます。
5. 赤い隕石に当たるとゲームオーバーです。
6. ゲームオーバー後は、画面をもう一度タップするとリトライできます。

## ファイル構成

- `index.html`: ゲーム画面とHUDの土台です。
- `style.css`: 画面全体、スコア表示、開始・終了メッセージの見た目です。
- `script.js`: タッチ操作、ゲームループ、当たり判定、スコア保存などの処理です。

## 技術メモ

- Canvas APIで2D描画しています。
- ゲームプレイ中の操作はボタンではなく、タッチまたはドラッグで行います。
- ベストスコアはブラウザの`localStorage`に保存されます。
